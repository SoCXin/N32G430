[N32G452](https://github.com/SoCXin/N32G452) 高达512KB 片内FLASH，144KB SRAM，多达18个数字通讯接口及4个模拟接口，内置10余种密码算法硬件加速引擎，支持存储加密、用户分区保护、安全启动等多种安全特性，可广泛应用于消费及工业类等应用领域。
